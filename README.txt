Файл запуска сервера находится:
Server\bin\Release

Параметры запуска:
[Data Base Path] - Путь к базе данных (Например ./WordsDict.db3)
[Server Port] - Порт, который будет прослушивать сервер (Например 7788)

Сервер использует базу данных SQLite

Команды сервера:
make-dict - создать словарь (отчищает текущий словарь, заносит новые слова по указанным в ТЗ отборам)
update-dict - дополняет словарь словами по указанным в ТЗ отборам
clear-dict - отчищает словарь

Файл запуска клиента находится:
Client\bin\Release

Параметры запуска:
[Host Name] - IPv4 адрес или наименование хоста (например 127.0.0.1 или localhost)
[Host Port] - Порт хоста (Например 7788)
